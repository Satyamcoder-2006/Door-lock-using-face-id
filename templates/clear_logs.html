Method 4: Manual file deletion

If none of the above methods work, you can manually delete the log files:

  1. Navigate to the application folder
  2. Look for files named access_logs.json
  3. Delete these files or replace their content with []
  4. Check both the main folder and any logs subfolder
Make sure the application is not running when you delete these files.
